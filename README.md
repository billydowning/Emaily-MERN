<h1>Emaily</h1>
Emaily is a survey application that allows a user to purchase credits, create a survey email, specify a list of recipients, and receive feedback on sent surveys.

<h2>Built with:</h2>
<ul>
  <li>React</li>
  <li>Redux</li>
  <li>Express</li>
  <li>Passport</li>
  <li>Stripe</li>
  <li>Semantic UI</li>
</ul>
